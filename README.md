Smart Expense Tracker

Smart Expense Tracker is a full-stack web application designed to help users record and analyze their daily expenses.

Live Demo
Frontend: https://extraordinary-hamster-a3e7af.netlify.app  
Backend: https://expense-tracker-backend-91nt.onrender.com

Features
- Add expenses with description and amount
- Automatic category prediction
- Rule-based category correction
- Real-time total spending calculation
- Pie chart analytics

Tech Stack
Frontend: React (Vite), Recharts  
Backend: Flask (Python)  
Deployment: Netlify & Render

Description
This application allows users to track expenses and visualize spending patterns through interactive charts.
